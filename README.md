Functionalities-

● Render Image gallery.

● Clicking on an image should open up the modal.

● Navigation arrows to move to the previous or next image.

● Images should be Lazy loaded.

● Should be responsive.

Bonus Points-

● Accessibility

● Vanilla js, Type safety.
